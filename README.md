# CTF 🚩

This repository contains my write-ups, notes, scripts, and solutions for various [Capture The Flag](https://en.wikipedia.org/wiki/Capture_the_flag#Computer_security) (CTF) challenges focused on cybersecurity topics.
The purpose of this repository is to document my learning process, methodologies, and technical approaches used during offensive security challenges and labs.

---

## ⚠ Disclaimer

All content in this repository is intended for:

- Educational purposes
- Ethical hacking practice
- Security research
- CTF environments

Do not use any techniques demonstrated here against systems you do not own or have explicit permission to test.

---

## 📘 Notes

Some write-ups may intentionally omit flags, credentials, or challenge-specific answers depending on platform rules and disclosure policies.
Write-ups may also be removed at the discretion of CTF organizers upon request.

---

## 🚀 Goals

This repository serves as:

- A personal knowledge base
- A portfolio of practical security skills
- Documentation of challenge-solving methodologies
- A reference for future research and learning
